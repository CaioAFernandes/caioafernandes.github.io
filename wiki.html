Sobre o Projeto Wiki Mangás


Sistema criado para gerenciar o armazenamento da localização e sittuação sobre mangás.

O desenvolvimento foi proposto como projeto na Fatec - ZL, e foi concretizado em parceria com Alicia Suemi Yoshida - Analista de Dados, Caio Alberto Fernandes - Full Stack e Suzana Azevedo Portilho Pinheiro - Front-End..

Informações Técnicas

Estrutura Tipo
Liguagem de Programação JEE
Banco de Dados SQL Server e MySQL
Frameworks Bootstrap
Gerenciador de Versões Github
