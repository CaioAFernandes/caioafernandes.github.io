Sobre

Projeto desenvolvido com objetivo de redimensionar e disponibilizar as imagens com o tamanho corrigido para dowonload sendo composto pelas seguintes características:
  • Envio de imagens
  • Redisionamento das imagens
  • Disponibilidade das imagens
  • Limpeza dos arquivo do servidor

Projeto desenvolvido para introdução a linguagem NodeJS e suas diversidades
